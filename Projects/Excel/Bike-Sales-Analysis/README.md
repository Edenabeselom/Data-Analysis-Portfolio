# Bike Sales Data Analysis

## 🚀 Project Overview
This project presents an interactive sales dashboard developed in Excel to analyze customer purchasing behavior. The primary goal was to identify key demographic and economic drivers that influence the decision to purchase a bicycle.

## 🛠 Key Features & Skills
*   **Data Cleaning:** Standardized datasets, corrected formatting, and handled missing values to ensure analytical accuracy.
*   **Data Analysis:** Utilized Pivot Tables to aggregate consumer insights across income, age brackets, and regional metrics.
*   **Data Visualization:** Designed an interactive dashboard featuring dynamic charts and slicers for real-time user filtering by occupation, commute distance, and purchase status.

## 📈 Key Insights
*   **Market Distribution:** North America represents the primary market, accounting for over 50% of the surveyed population.
*   **Purchase Behavior:** Nearly half of the total population (48.2%) opted to purchase a bike, offering significant room for targeted marketing efforts.
*   **Economic Segmentation:** The average income of $56,360 highlights a middle-to-high income customer base, which serves as a core target for premium bike models.

## 📁 Project Structure
*   `Bike Sales Dashboard.xlsx`: The raw dataset and the interactive dashboard.
*   `DATA_DICTIONARY.md`: Definitions of the dataset columns for data governance.
