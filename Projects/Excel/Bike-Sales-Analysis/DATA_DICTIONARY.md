# Data Dictionary: Bike Sales Analysis

This document provides definitions for the columns used in the Bike Sales dataset to ensure clarity and transparency in our analysis.

| Column Name | Description | Data Type |
| :--- | :--- | :--- |
| **ID** | Unique identifier for each individual record. | Numeric |
| **Marital Status** | Marital status of the individual (Married/Single). | Categorical |
| **Gender** | Gender of the individual (Female/Male). | Categorical |
| **Income** | Reported annual income of the individual in USD. | Numeric |
| **Children** | Number of children the individual has. | Numeric |
| **Education** | Highest level of education attained. | Categorical |
| **Occupation** | Current professional industry or role. | Categorical |
| **Home Owner** | Indicates if the individual owns their home (Yes/No). | Categorical |
| **Cars** | Number of cars owned by the individual. | Numeric |
| **Commute Distance** | The distance traveled by the individual to work. | Categorical |
| **Region** | Geographic region of the individual (North America, Europe, Pacific). | Categorical |
| **Age** | Age of the individual in years. | Numeric |
| **Purchased Bike** | Target variable indicating if a bike was purchased (Yes/No). | Categorical |
| **Age Brackets** | Grouped age ranges for demographic analysis. | Categorical |
| **Income Group.** | Income levels categorized into ranges for segmentation. | Categorical |
