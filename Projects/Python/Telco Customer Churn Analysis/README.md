# Telco Customer Churn Analysis

This repository contains a data analysis project focused on understanding customer behavior and the factors contributing to service churn within the telecommunications industry.

## 📊 Project Overview
This analysis investigates churn patterns across different customer segments, specifically focusing on service subscriptions. The goal is to identify key drivers that correlate with customer attrition to support better retention strategies.

## 📂 Data Summary
The analysis utilizes a dataset containing customer demographic, account, and service information. Key metrics calculated include:
*   **Total Customer Reach**: The total number of customers analyzed in the dataset.
*   **Churn Rate**: The percentage of customers who have churned ('yes').
*   **Service Profiles**: Insights into how service subscriptions, particularly **Internet Service Type**, relate to retention.

## 🛠 Analysis Results
### Churn Rate by Internet Service Type
The analysis reveals that different internet service technologies have significantly different churn rates. Users on Fiber optic connections demonstrate a higher propensity to churn compared to those using DSL or those with no internet service, highlighting a key area for service quality improvement or targeted retention efforts.
