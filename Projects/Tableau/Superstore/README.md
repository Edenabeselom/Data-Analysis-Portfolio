# Superstore Sales Analysis

This repository contains a comprehensive data analysis of the "Sample - Superstore" dataset, visualized using Tableau.

## 📊 Project Overview
This project provides a business intelligence dashboard to track sales performance, identify regional profitability, and analyze sub-category trends. The goal is to provide actionable insights into revenue, profit, and the health of the business through various metrics.

## 📂 Data Dictionary
The dataset (`Sample - Superstore.csv`) consists of the following key dimensions and measures:

| Category | Field Name | Description |
| :--- | :--- | :--- |
| **Location** | `City`, `State`, `Region` | Geographic breakdown of transactions |
| **Time** | `Order Date`, `Ship Date` | Temporal tracking of orders |
| **Product** | `Category`, `Sub-Category` | Product hierarchy details |
| **Financial**| `Sales`, `Profit`, `Discount` | Core performance measures |

## 🛠 Technical Details
*   **Analysis Tool**: Tableau (Version 18.1)
*   **Operating Environment**: Windows
*   **Build Version**: 2026.2.0 (20262.26.0603.1643)

## 📈 Key Metrics
*   **Total Revenue**: Aggregated sum of `Sales`.
*   **Total Profit**: Aggregated sum of `Profit`.
*   **Profit Margin**: Calculated as $\frac{\sum \text{Profit}}{\sum \text{Sales}}$.
